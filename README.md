# Tic Tac Toe AI

A console-based Tic Tac Toe game developed in Python, featuring multiple AI strategies including Random AI, Smart AI, and Minimax AI. The project is built using NumPy and follows a clean, modular structure for better organization and scalability.
This game allows players to compete either against another player or against different types of AI opponents. Each AI mode provides a different level of difficulty, with the Minimax algorithm offering an optimal and unbeatable gameplay experience.

## Features

- Player vs Player mode  
- Player vs Random AI  
- Player vs Smart AI (rule-based logic)  
- Player vs Minimax AI (optimal strategy)  
- Clean terminal-based interface  
- Modular code structure with separate files  

## 🧠 AI Strategies

- **Random AI**: Selects moves randomly from available spots  
- **Smart AI**: Attempts to win or block the opponent using basic logic  
- **Minimax AI**: Uses the Minimax algorithm to always choose the best possible move

## 🛠️ Requirements

- Python 3.x  
- NumPy  

Install NumPy using:
pip install numpy


## ▶️ How to Run

Clone this repo and open the command prompt and go to the folder where all files are clonedthen run the command in cmd  
python main.py


## 🎯 How to Play

- Choose a game mode at the start  
- Input row and column values between 0 and 2  
- Try to win against the opponent or AI
